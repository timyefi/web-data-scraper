# 通用网站数据抓取方法论

## 网站交互模式分类

### 类型1：搜索-列表-详情型
**特征**：搜索框 + 筛选条件 + 结果列表 + 分页 + 详情页/下载链接
**适用**：巨潮资讯、中国货币网、政府采购网、上交所、深交所

**标准操作流程**：
1. 定位搜索框（通常为 `input[type=text]` 或 `textbox` 角色）
2. 识别筛选条件（下拉 `combobox`、日期选择器、分类Tab）
3. 点击查询按钮（`button` 或 `link "查询"`）
4. 等待结果加载（snapshot确认）
5. 提取列表数据（标题、日期、链接）
6. 处理分页
7. 进入详情页或构造下载链接

### 类型2：导航-表格型
**特征**：分类导航树 + 指标选择 + 数据表格/图表
**适用**：国家统计局、央行数据、Wind终端Web版

**标准操作流程**：
1. 展开分类导航树
2. 点击目标指标
3. 等待数据表格加载
4. 提取表格数据（可能需要处理Echarts等可视化组件）
5. 查找导出按钮（CSV/Excel下载）

### 类型3：API直取型
**特征**：有公开或半公开的JSON API
**适用**：巨潮资讯、中国货币网

**标准操作流程**：
1. 通过 network 监听发现API端点
2. 分析请求参数格式
3. 建立 session（有些API需要先访问页面获取cookie）
4. 构造请求参数
5. 发送请求获取JSON
6. 解析并转换为CSV/Markdown

## 元素识别策略

### Playwright Snapshot 中的关键角色

| ARIA角色 | 含义 | 常见用途 |
|----------|------|---------|
| `textbox` | 文本输入框 | 搜索框、关键词输入 |
| `combobox` | 下拉选择 | 年份、类型、板块筛选 |
| `button` | 按钮 | 查询、重置、导出 |
| `link` | 链接 | 详情页、下载链接、分页 |
| `table` | 表格 | 数据列表 |
| `listitem` | 列表项 | 搜索结果项 |
| `tab` | 标签页 | 分类切换 |
| `treeitem` | 树节点 | 分类导航 |

### 识别技巧

1. **搜索框**：找 `textbox` + `placeholder` 含"搜索/查询/请输入"
2. **下拉选择**：找 `combobox`，用 `select` 命令操作
3. **查询按钮**：找包含"查询/搜索/确定"文本的 `button` 或 `link`
4. **结果列表**：找重复的 `listitem` 或 `row`
5. **下载链接**：找 `link` + URL 含 `.pdf` / `download` / `contentId`
6. **分页**：找包含"下一页/下页/Next"的 `link` 或 `button`
7. **导出按钮**：找包含"导出/下载/Export"的 `button`

## 数据加载方式判断

### 判断方法

1. **打开页面后立即 snapshot**：
   - 如果能看到完整数据 → 静态HTML渲染
   - 如果看到加载动画/空区域 → 动态加载

2. **使用 network 监听**：
   ```bash
   "$PWCLI" network
   # 查看是否有 XHR/Fetch 请求返回 JSON
   ```

3. **查看页面源码**：
   ```bash
   "$PWCLI" eval "document.documentElement.outerHTML.substring(0, 500)"
   # 如果数据已在HTML中 → 静态渲染
   ```

### 三种加载方式及对应策略

| 加载方式 | 特征 | 推荐策略 |
|---------|------|---------|
| 静态HTML | 页面源码包含数据 | 直接eval提取 |
| AJAX动态 | network有JSON请求 | 拦截API直接调用 |
| JS渲染 | 数据在JS变量中 | eval提取JS变量 |

## 分页处理策略

### 传统分页（页码按钮）
```
loop:
  snapshot → 提取当前页数据
  查找"下一页"按钮
  if 存在 → click下一页 → 继续loop
  else → 结束
```

### 无限滚动
```
loop:
  snapshot → 提取当前可见数据
  mousewheel 或 press End → 滚动到底部
  等待新数据加载
  if 数据量不再增加 → 结束
```

### API分页
```
loop (pageNo = 1, 2, 3, ...):
  构造请求参数 pageNum=pageNo
  发送API请求
  if 返回数据为空 → 结束
  提取数据
```

## 输出格式规范

### Markdown格式
```markdown
# [网站名称] 查询结果

**查询时间**：2024-01-15 14:30:00
**查询条件**：关键词=XXX, 类型=XXX, 时间范围=XXX

| 序号 | 标题 | 日期 | 链接 |
|------|------|------|------|
| 1 | ... | ... | ... |
```

### CSV格式
```csv
序号,标题,日期,链接,来源
1,公告标题,2024-01-15,https://...,网站名
```

### 下载任务配置格式
```json
{
  "output_dir": "./downloads/",
  "tasks": [
    {
      "name": "文件显示名",
      "url": "https://example.com/file.pdf",
      "output_path": "文件名.pdf",
      "retries": 3
    }
  ]
}
```
